# Smart Attendance System

A face recognition-based automated attendance system that marks attendance during scheduled lectures using real-time facial detection.

## Features

- ✅ **Automated Attendance**: Real-time face recognition
- ✅ **Lecture Scheduling**: Time-based attendance restrictions  
- ✅ **Student Management**: Easy registration with single photo
- ✅ **Comprehensive Reports**: Filter by lecture, student, or date
- ✅ **Data Management**: Flexible record management

## Installation

```bash
# Install dependencies
pip install face-recognition opencv-python pillow

# Run the system
python gui.py
